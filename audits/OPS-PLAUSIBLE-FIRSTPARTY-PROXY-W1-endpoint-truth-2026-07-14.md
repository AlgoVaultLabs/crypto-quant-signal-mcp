- upstream = `127.0.0.1:8000` (local CE), not `https://plausible.io` — no upstream-Host rewrite needed for a local proxy.
- **event route MUST forward the real client IP** for CE bot-filtering + geo. Apex is CF-orange → CF sends `CF-Connecting-IP: <real>` (+ XFF). Set `header_up X-Forwarded-For {http.request.header.CF-Connecting-IP}` (+ keep `Host`) on the event route. The existing `plausible.algovault.com` vhost relies on Caddy's default XFF-append; be explicit here since the apex adds a hop.
- script route: `rewrite * /js/pa-gfOGxVcQKNfocPr5V3vR8.js` → `reverse_proxy 127.0.0.1:8000` (serve the CE script first-party); cache-eligible.
